#### О прокете

Основная цель данного проекта — создание и контроль задач с назначением определённым группам лиц.  
Приложение получилось универсальным, что позволяет использовать его в разных командах.  
Для разделения задач используется **Kanban‑доска**.

#### Возможности

Приложение имеет клиент-серверную архитектуру и разделение на три роли:

- **Admin** — администратор системы. В его задачи входит управление пользователями 
  (бан / разбан), создание и редактирование задач, создание пользователей и
   т.д. Admin имеет полный контроль над системой.

- **Manager** — формирует и редактирует задачи, а также удаляет их.

- **Worker** — сотрудник, который выполняет определённый список задач.

#### Установка

##### Клиенская часть

Клонируйте репозиторий:

```bash
git clone https://github.com/a-android-a/TaskForge

```

```bash
cd Client
mkdir build
cd build
cmake .. 
make
```

Перейдите в папку клиента и соберите проект:

Скопируйте файлы стилей (или создайте свои):

```bash
cp -r ../style .
```

Сгенерируйте ключ и сертификат с помощью OpenSSL:

```bash
openssl req -x509 -newkey rsa:2048 -keyout server.key -out server.crt -days 365 -nodes
```

**ВАЖНО НЕ ХРАНТИЕ server.key НА КЛИЕНТЕ**

Также не забудьте указать IP-адрес вашего сервера в создание сертефиката

##### Серверная часть

Перейдите в папку сервера и соберите проект:

Создаём папку build

```bash
mkdir build
cd build
cmake .. 
make
```

Скопируйте базу данных:

```bash
cp -r DB build
```

Создайте файл конфигурации `config.json` (пример структуры см. ниже).

```json
{
    "appearance": {
        "language": "ru",
        "stylePath": "style/darkStyle.qss",
        "theme": "light"
    },
    "server": {
        "certificate": "server.key",
        "host": "127.0.0.1",
        "port": 4433
    },
    "user": {
        "defaultName": "333"
    }
}
```

Если вы сделали всё правилньно то увидите в терминале Server Start без каких либо ошибок для входа в админ панель используйте логин: admin пароль: admin
